RInstalling JSON Server RG:-

--npm install -g json-server--

RCheck JSON Server version RG:-

--json-server -v --

RStep 1: Create ADDDB.json file.

   touch ADDDB.json
   cat ADDDB.json
   nano ADDDB.json
   cat ADDDB.json

RStep 2: Run JSON server.This will create REST API for ADDDB
-- json-server--watch ADDDB.json --

Rstep 3: Get all Entries using curl command
-- curl -X GET -H "Content-Type:application/json" "http://localHost:4000/Addressbook/list" --

RStep 4: Adding New Entry Lisa.Here We notice HTTP Method POST is used as well as using-d option JSON is passed in the body

--- curl -X POST-H "Content-Type:application/json" -d'{"name":"Lisa","contact":"9627974345"}' "http://localhost:4000/Addressbook/create" ---

RStep 5: for Update a contact of Lisa.

--- curl -XPUT -H "Content-Type:application/json" -d'{"name":"Lisa","contact":"9837694080"}' "http://localhost:4000/Addressbook/update/4" ---

RStep 6: For Deleting Records of Lisa.

--- curl -XDELETE -H "Content-Type:application/json" "http://localhost:4000/Addressbook/delete/4" ---
